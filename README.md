# Red Team Engagement Lab

This repository documents a full end-to-end Red Team engagement
conducted in a controlled and authorized lab environment.

## Scope
- OSINT & Reconnaissance
- Phishing-based Initial Access
- Vulnerability Exploitation
- Lateral Movement & Persistence
- Post-Exploitation & Exfiltration
- Blue Team Detection Review
- Final Reporting

## Environment
- Kali Linux (Attacker)
- Windows VM (Victim)
- Metasploitable2/3
- Isolated Host-Only Network

## Disclaimer
All activities were performed in an isolated lab using test systems
and dummy credentials strictly for educational purposes.
